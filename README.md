## 🚀 Getting Started
To explore or run any of these projects locally, follow these steps:

Clone the repository:

git clone git@github.com:sycolon/py_examples.git
cd py_examples


Navigate to a specific project:

cd messengerApp

Install dependencies (if any):
Most projects will require only standard Python. However, if a requirements.txt file is present in a project's directory, install its dependencies:

pip install -r requirements.txt

Run the project:
Execute the main/App Python file for the project:

python MessengerApp.py

(Or python script.py depending on the project's entry point.)
